Rufus

Δημιουργήστε USB με δυνατότητα εκκίνησης εύκολα και γρήγορα

Το Rufus είναι ένα πρόγραμμα που σας βοηθά με τη διαμόρφωση και τη δημιουργία USB με δυνατότητα εκκίνησης, όπως USB keys /pendrives, κάρτες μνήμης, κτλ.

Μπορεί να είναι ιδιαίτερα χρήσιμο για τις περιπτώσεις όπου:

  • θέλετε να δημιουργήσετε μέσα εγκατάστασης USB από είδωλα ISO με δυνατότητα εκκίνησης (Windows, Linux, UEFI, κλπ.)
  • θέλετε να δουλέψετε σε ένα σύστημα το οποίο δεν έχει εγκατεστημένο κάποιο λειτουργικό σύστημα
  • θέλετε να κάνετε flash το BIOS ή άλλο υλικολογισμικό από το DOS
  • θέλετε να εκτελέσετε ένα βοηθητικό πρόγραμμα(low-level)

Παρά το μικρό μέγεθος της εφαρμογής, το Rufus παρέχει οτιδήποτε χρειάζεστε!

Ένα πλεονέκτημα του Rufus είναι ότι είναι αρκετά γρήγορο. Παραδείγματος χάριν, είναι περίπου δύο φορές πιο γρήγορο απο τα UNetbootin, Universal USB Installer ή το Windows 7 USB download tool, στην δημιουργία ενός USB εγκατάστασης των Windows 7 από ένα είδωλο ISO. Επίσης το Rufus είναι οριακά πιο γρήγορο και στις περιπτώσεις δημιουργίας USB με δυνατότατα εκκίνησης για Linux από ένα είδωλο ISO. (1)
Ο κατάλογος των μορφών ISO που υποστηρίζονται απο το Rufus μπορεί να βρεθεί στο κάτω μέρος της σελίδας. (2)

Θα επωφεληθώ αυτής της ευκαιρίας για να εκφράσω την ευγνωμοσύνη μου προς τους μεταφραστές που κατέστησαν δυνατό για το Rufus, καθώς και για την ιστοσελίδα να μεταφραστεί σε διάφορες γλώσσες. Αν διαπιστώσετε ότι μπορείτε να χρησιμοποιήσετε το Rufus στη δική σας γλώσσα, θα πρέπει να τους ευχαριστήσετε πραγματικά!

Χρήση

Κάντε λήψη του αρχείου για να το εκτελέσετε – δε χρειάζεται κάποια εγκατάσταση.

Το εκτελέσιμο αρχείο είναι υπογεγραμμένο ψηφιακά και η υπογραφή πρέπει να αναφέρει:

  • "Akeo Consulting" (v1.3.0 ή αργότερη)
  • "Pete Batard - Open Source Developer" (v1.2.0 ή νωρίτερη)

Σημειώσεις σχετικά με την υποστήριξη του DOS:

Εάν δημιουργήσετε ένα DOS δίσκο με δυνατότητα εκκίνησης και χρησιμοποιήσετε ένα πληκτρολόγιο μη-ΗΠΑ, το Rufus θα επιχειρήσει να επιλέξει μια διάταξη πληκτρολογίου σύμφωνα με τις τοπικές ρυθμίσεις του συστήματός σας. Σε αυτή την περίπτωση, το FreeDOS, είναι η προεπιλεγμένη επιλογή και συνιστάται αντί του MS-DOS διότι υποστηρίζει περισσότερες διατάξεις πληκτρολογίου.

Σημειώσεις σχετικά με την υποστήριξη ειδώλων ISO:

Όλες οι εκδόσεις του Rufus από v1.1.0 και μετά επιτρέπουν τη δημιουργία ενός USB με δυνατότητα εκκίνησης από ένα είδωλο ISO (.iso).

Η δημιουργία ενός ειδώλου ISO από ένα φυσικό δίσκο ή από ένα σύνολο αρχείων είναι πολύ εύκολο να πραγματοποιηθεί. Παρόλα αυτά, μόνο μέσω της χρήσης μιας εφαρμογής εγγραφής CD, όπως τα ελεύθερα διαθέσιμα προγράμματα CDBurnerXP ή ImgBurn.

Δωρεές

Επειδή λαμβάνω αυτή την ερώτηση σε τακτική βάση, δεν υπάρχει κουμπί δωρεάς σε αυτή τη σελίδα.

Ο κύριος λόγος είναι ότι αισθάνομαι πως το σύστημα δωρεών δεν βοηθά πραγματικά την ανάπτυξη λογισμικού, αλλά χειρότερα μπορεί να προκαλέσει ενοχές στους χρήστες που επιλέγουν να μη δωρίσουν.

Αν πραγματικά επιμένετε, μπορείτε πάντα να κάνετε μια δωρεά στο Ίδρυμα Ελεύθερου Λογισμικού, δεδομένου ότι είναι ο κύριος λόγος που λογισμικό όπως το Rufus είναι διαθέσιμο.

Εν πάση περιπτώσει, θα επωφεληθώ αυτής της ευκαιρίας για να σας ευχαριστήσω για τη συνεχή υποστήριξη και τον ενθουσιασμό σας σχετικά με αυτό το πρόγραμμα: το εκτιμούμε πάρα πολύ!

Αλλά σας παρακαλώ να συνεχίσετε να αισθάνεστε ελεύθεροι να χρησιμοποιείτε το Rufus χωρίς καμία ενοχή για τη μη οικονομική συνεισφορά σε αυτό – δεν ειναι απαιτούμενη απο εσάς!
